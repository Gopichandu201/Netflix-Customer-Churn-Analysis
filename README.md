# Netflix-Customer-Churn-Analysis
This project analyzes Netflix customer churn using data analytics techniques. The goal is to identify patterns in customer behavior and understand factors that influence subscription cancellations. The insights can help businesses improve customer retention strategies.

# Tools used
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (MySQL)
Power BI
GitHub

# Data set
The dataset contains information about Netflix customers including:

Age and Gender
Subscription Type
Watch Hours
Last Login Days
Region and Device
Monthly Fee
Favorite Genre
Customer Churn Status

# Project Workflow
Data Cleaning and Preprocessing using Python
Exploratory Data Analysis (EDA) to understand customer behavior
SQL queries for data analysis
Power BI dashboard creation for visualization

# Key Insights
Customers with low watch hours are more likely to churn.
Customers who haven't logged in recently have a higher churn rate.
Subscription type impacts customer retention.
Viewing preferences vary by region and device.

# Dashboard Features 
The Power BI dashboard includes:
Total Customers
Total Churn
Active Customers
Churn Rate
Customer Distribution by Region
Churn by Subscription Type
Favorite Genres Analysis
Customer Details Table
